# DummyRestTesting
Public Git to test [DummyApi](http://dummy.restapiexample.com/). RestFULLy.

# Dependencies
 - **Java** -> openjdk 11.0.4 
 - **Maven** -> Apache Maven 3.6.0
 
# Getting Started
 - `clone git`
 - `mvn test`
